Hello!!!
Ariel is supposed to move onto land with Prince Eric tomorrow but misplaced her things :OOOO
Help Ariel find her missing shells by maneuvering her through thr grotto and collectecting her 4 missing shells.
Be careful!!! Due to underwater palace renovations there is some obnoxious and poisonous sulphur bubbles floating around!
This is expecially the case in Ariel's secret hidden grotto of knick knacks!

Controls are 

Keyboard
-----------------
SELECT -> backspace
START -> enter
A -> z
B -> x
L -> a
R -> s

you can use you up, down, left, and right keys to maneuver Ariel!
to make type "make mgba" into folder with game within command 


Happy Playing!
